Geokode

A fast, accurate, self-hosted geocoding service written in Rust. Forward & reverse geocoding, autocomplete, batch processing.

Rust Self-Hosted FST + R-tree REST API Docker Ready AGPL-3.0-or-later

Features

Forward Geocoding

Text query → coordinates. Fuzzy matching with abbreviation expansion. FST-powered prefix search for sub-millisecond lookups.

Reverse Geocoding

Coordinates → nearest address. R-tree spatial index with k-nearest-neighbor queries. Confidence scoring based on distance.

Autocomplete

Interactive prefix search for UIs. Returns ranked results as you type with spatial bias support.

Batch Processing

Geocode thousands of addresses in a single request. JSON API for bulk operations.

Address Parsing

Structured decomposition: house number, street, city, state, postal code, country. Street abbreviation normalization.

Multiple Data Sources

Ingest OpenAddresses CSV, GeoJSON, and custom formats. Your data stays local — no external API dependencies.

Docker Ready

Multi-stage Dockerfile, docker-compose with health checks. Deploy in minutes.

High Performance

Pure Rust with zero-copy parsing. FST for O(1) text lookup, R-tree for spatial queries. Handles millions of records.

GeoLang Ecosystem

Integrates with ETL pipelines for address enrichment, stream processors for GPS context, and Ptolemy for versioned storage.

API Endpoints

GET /forward?q=123+Main+St
Forward geocode — text to coordinates
GET /reverse?lon=-89.65&lat=39.78&limit=5
Reverse geocode — coordinates to nearest addresses
GET /autocomplete?q=main&limit=10
Autocomplete — prefix search for interactive UIs
POST /batch
Batch forward geocode — multiple addresses in one request
GET /health
Health check — service status and record count

Quick Start

# Using Docker
docker compose up -d

# Or build from source
cargo build --release -p geokode-cli

# Start server with OpenAddresses data
geokode serve -d addresses.csv --bind 0.0.0.0:3000

# Forward geocode
curl "http://localhost:3000/forward?q=123+Main+St,+Springfield"

# Reverse geocode
curl "http://localhost:3000/reverse?lon=-89.65&lat=39.78"
